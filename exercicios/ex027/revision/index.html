Aula 01 - Primeiro Passos com Flexbox CSS

Visualizando a estrutura das Flexbox, temos dois elementos importates: o pai, que é o contêiner, e o filho, que é o item que está dentro desse contêiner; podemos chamá-los, respectivamente, de flex-container e flex-item.

Para que o Flexbox seja utilizado, declaramos display: flex; na caixa pai.

Aula 2 - Direções e Eixos Flexbox CSS

A propriedade flex-direction diz se iremos trabalhar com linhas ou colunas como o eixo principal. Ela pode ter os seguintes valores:

  • row (valor padrão), onde os itens são colocados em linha, da esquerda para a direita;
  • row-reverse, onde os itens são colocados em linha, da direta para esquerda;
  • column, onde os itens são colocados em coluna, de cima para baixo;
  • column-reverse, onde os itens são colocados em coluna, de baixo para cima;

Trabalhando com eixos (axis), nós temos:

  • O main-axis, que é o eixo principal, possuindo dois pontos: o main-start e o main-end;
  • O cross-axis, que é o eixo transversal, possuindo dois pontos: o cross-start e o cross-end.

Cada flex-direction tem uma disposição diferente dos eixos.

Quando não definimos, temos por padrão a propriedade flex-wrap com o valor nowrap; ou seja, para não encapasular. Quando 'apertamos' essa caixa, ela aperta os itens. Obs.: o tamanho de cada item vai depender muito do que ele seja; a não ser que você tenha definido previamente.

O valor wrap fará com que haja uma quebra para que o item continue com seu mesmo tamanho, mas dentro do contêiner. Ele sempre irá quebrar na direção do cross-axis.

O valor wrap-reverse fará com que a quebra dos itens seja para cima.

A propriedade flex-flow é um shorthand que une flex-direction e flex-wrap. Exemplo: flex-flow: row nowrap;.

Quando você utiliza um flex-wrap e deixa os itens com um flex: auto;, eles começam a quebrar quando dá o tamanho do conteúdo dentro dos itens.

Aula 04 - Alinhamento nos Eixos Flexbox

A propriedade justify-content faz o alinhamento em relação ao main-axis:

  • Com o valor flex-start, o primeiro item fica grudado no main-start;
  • Com o valor flex-end, o último item fica grudado no main-end;
  • Com o valor center, todos os itens ficam centralizados de acordo com o main-axis na caixa;
  • Com o valor space-between, ele vai grudar o primeiro item no main-start, o último item no main-end e os outros ficam divididos no meio com espaços iguais de distância;
  • Com o valor space-evelyn, esses mesmos 3 itens dentro da caixa ficarão separados pela mesma distância de um para outro e a mesma distância para o main-start e o main-end; ou seja, todos os espaços ficam do mesmo tamanho;
  • Com o valor space-around, ele vai contar quantos itens tem dentro da caixa e vai dividir espaços na quantidade deles; dentro de cada espaço, fica o item centralizado;

Aula 06 - Alinhamento de Elementos Empacotados no Flebox

É utilizado no conteúdo que se 'quebra'.

A propriedade align-content é diferente da align-items; a segunda alinha elementos com base no cross-axis, enquanto a primeira alinha os elementos com base nos espaços que ficaram disponíveis e com base no cross-axis da caixa pai.

Ele funciona quando o flex-wrap está com o valor wrap, ou seja, para quebrar. Os valores que o align-content pode receber são:

  • stretch: ele pega a dimensão inteira e divide na quantidade de linhas, caso seja 'row', coloca cada linha em seus espaços e estica eles do start ao end do espaço que ele está;
  • flex-start: ele fica grudado no início da parte; mas ele não tem as partes divididas, como no stretch; ele só deixa o espaço em branco embaixo;
  • flex-end: ele fica grudado no final da parte; deixa a parte em branco encima;
  • center: ele tem um espaço em branco em cima e embaixo de mesmo tamanho para que as linhas fiquem centralizadas;
  • space-between: ele pega os primeiros elementos e coloca no cross-start e pega os últimos e deixa no cross-end; os outros elementos ficam divididos no meio;
  • space-evenly: cria espaços em branco de mesmo tamanho ao lado de todos elementos;
  • space-around: divide todo o eixo transversal na quantidade de linhas, coloca os elemetnos em cada um desses novos espaços e centraliza eles dentro desses espaços;

Aula 07 - Anatomia dos Itens Flexbox

Dentro dos flex-container temos os flex-items; a propriedade de itens que vimos até agora é a flex: auto;.

A propriedade order recebe valores numéricos; ela organiza os itens os colocando em ordem, com base na numeração, de forma crescente.

A propriedade align-self se aplica aos itens e funciona no cross-axis; seu valores são:

  • auto: ele herda a característica de alinhamento do elemento pai;
  • flex-start: fica grudado na parte de cima;
  • flex-end: fica grudado na parte de baixo;
  • center: fica centralizado;
  • stretch: estica para fazer o preenchimento de todo cross-axis.
