Aula 01 - Primeiro Passos com Flexbox CSS

Visualizando a estrutura das Flexbox, temos dois elementos importates: o pai, que é o contêiner, e o filho, que é o item que está dentro desse contêiner; podemos chamá-los, respectivamente, de flex-container e flex-item.

Para que o Flexbox seja utilizado, declaramos display: flex; na caixa pai.

Aula 2 - Direções e Eixos Flexbox CSS

A propriedade flex-direction diz se iremos trabalhar com linhas ou colunas como o eixo principal. Ela pode ter os seguintes valores:

  • row (valor padrão), onde os itens são colocados em linha, da esquerda para a direita;
  • row-reverse, onde os itens são colocados em linha, da direta para esquerda;
  • column, onde os itens são colocados em coluna, de cima para baixo;
  • column-reverse, onde os itens são colocados em coluna, de baixo para cima;

Exemplos na prática:

Trabalhando com eixos (axis), nós temos:

  • O main-axis, que é o eixo principal, possuindo dois pontos: o main-start e o main-end;
  • O cross-axis, que é o eixo transversal, possuindo dois pontos: o cross-start e o cross-end.

Cada flex-direction tem uma disposição diferente dos eixos.
